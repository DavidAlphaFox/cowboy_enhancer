Adds new data in map format to the session data of a session represented by a SessionID argument.

Returns: {error, {already_exist_data, ExistingDataMap} if any key in NewSessionDataMap is already in session data.

Example: 
    add_session_data(SessionID, #{age => 31, address => "Highway"}).
Note:
  • This function will not reset the expiration time countdown of the session. You can use refresh_session_expiration_time/1 after adding all data to reset session expiration time countdown.

create_session/2

create_session(SessionDataMap, ExpiredCallback) -> {ok, SessionID}

  • SessionDataMap = #{}
  • ExpiredCallback = function()
  • SessionID = session_id()

Creates a session with a generated unique id and a specified SessionDataMap in map format using the expired callback option.

ExpiredCallback is a function that will be called once the session expires and it must take an argument that is the expired Session

The return value of the expired callback controls the flow of the expiration process as follow:
  • {cancel, default}: cancels the expiration process and refresh the expiration time with the default value set in the app.config file under session_manager tag.
  • {cancel, NewExpirationTime}: does the same with the exception of a new expiration time will be used to refresh the expiration time of the session.
  • any() any other value will cause the session to be expired and deleted.
Example: 
    create_session(#{user_id => 1, username => "John"},
                   fun(ExpiredSession) -> {cancel, 1000} end).

See also: config_manager:session_manager_config/0.

ensure_session/1

ensure_session(SessionID) -> ok | {error, no_session} | {error, session_expired}

  • SessionID = session_id()

Does the same of check_session/1 but also refreshes the expiration countdown of the session.

Note:
    This function does 3 steps in one.
  • Checks whether the session is not expired or doesn't exists.
  • Forces garbage collection if the session is already expired.
  • Refreshes the expiration countdown of the session, so the session will not expire while this function is called.

See also: check_session/1.

get_session(SessionID) -> {ok, Session} | {error, no_session}

  • SessionID = session_id()
  • Session = session_map()

Gets the entire session object as a map by a SessionID argument.

With this you can get the session expire time and expiration callback function also the session data.

This function is equivalent to: 
    get_session(SessionID, [{result_format, map}]),
Example: 
    {ok, #{
         id := Id,
         session_data := SessionData,
         expired_callback := ExpiredCallback,
         expire_time := ExpireTime} = get_session(SessionID).
Note:
  • SessionData could be no_data if there is no session data.

See also: get_session/2, get_session_data/1, get_session_data/2.

  • SessionID = session_id()
  • NewSessionDataMap = #{}

Updates the session data of a session represented by a SessionID argument.

NewSessionDataMap must be a map with all already existing data keys with new values. All nonexistent keys will be ignored.

Example: 
    update_session_data(SessionID, #{username => "Tim"}).
Note:
  • This function will also reset the expiration time of the session. If you don't want this default behavior to happen you can use update_session_data_wrt/2 function.
