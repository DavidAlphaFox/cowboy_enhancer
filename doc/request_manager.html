map_request_params/2

map_request_params(ParamNames, Req) -> #{} | undefined

  • ParamNames = [atom(), ...]
  • Req = cowboy_req:req()

Parses the requested url params into a map(). NOTE: url params should be in the following format: http://host/handler-name/action-name[/param_name-param_value[-param_value_n]/...] i.e.: url: ".../id-1". #{id := ID} = map_request_params([id], Req).

url: ".../blog_id-1/blogpost_id-5". #{blog_id := BlogID, blogpost_id := BlogPostID} = map_request_params([blog_id, blogpost_id], Req).

NOTE: 'undefined' will be returned for the value of the param names that doesn't exists in the request url.
